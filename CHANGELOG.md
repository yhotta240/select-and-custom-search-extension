# Changelog

All notable changes to this project will be documented in this file.

## [1.5.2] - 2026-05-24

### Fixed

- 検索ボックスがページCSSの影響を受けないように修正
- ダークテーマ時のホバー色とテーマ切替ボタンのスタイルを調整

## [1.5.1]

### Added

- デフォルトの検索サイトリストに claude.ai を追加

### Fixed

- デフォルトの ChatGPT の検索クエリを修正

### Improved

- ポップアップの一部UI/UX改善

## [1.5.0]

### Added

- 検索機能にURL末尾対応を追加
- サイトの表示/非表示機能の追加

## [1.4.0]

### Improved

- 登録サイトリストの編集・削除機能の改善
- サイトの上書き登録，日本語キーワードに対応
- URL入力フォームの改善

## [1.3.0]

### Improved

- ポップアップのUI/UX改善
- 登録サイトリストの並び替え機能を追加

## [1.2.0 - 1.2.1]

### Improved

- テーマ変更，リスト表示ボタンにツールチップを追加
- フッターにメッセージパネルとリサイズ機能を追加
- サイト名をツールチップで表示

## [1.1.0 - 1.1.1]

### Improved

- ポップアップの幅に関するスタイルを修正
- OFFボタンにツールチップを追加
- 新しいタブで開くボタンを追加

## [1.0.1 - 1.0.8]

### Added

- ストアページへのリンクなどを追加
- アイコンの折り返しオプションを追加
- 選択したテキストからの距離を保存する機能を追加
- 検索モード（新しいウィンドウ，シークレットウィンドウ）を追加
- 検索アイコンの表示位置からの距離を設定する機能を追加
- テーマ変更機能を追加
- プレビューでアイコンの配置や個数を変更する機能を追加

### Changed

- YouTubeの検索に対応
- 検索モードと配置のUIを改善
- 検索サイトのリストをデフォルト設定として管理するように変更

## [1.0.0]

### Chore

- 初回リリース
